# インプット入力タブの操作方法について

インプット入力タブで提供している機能は次の通りです。

* VSCodeの起動

## VSCodeの起動

VSCode起動ボタンをクリックするとVSCodeが起動します。起動するVSCodeはパスタブで入力したVSCodeのフォルダパスにもとづき起動します。

![image-20240330163439509](./assets/image-20240330163439509.png)

VSCodeからインプットファイルを作成してください。
